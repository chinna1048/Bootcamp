Absolute path

  1. Start at the root directory ( / ) and work down.
		
  2. Write a slash ( / ) after every directory name (last one is optional)

Relative path

Relative path is defined as the path related to the present working directly(pwd).It starts at your current directory and never starts with a /

  1. .(a single dot) - this represents the current directory.
  2. ..(two dots) - this represents the parent directory.
